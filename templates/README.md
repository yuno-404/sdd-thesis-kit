# Templates

這個資料夾放可複用模板，不放正式稿。

- `templates/chapters/`：章節起稿模板

使用方式：

1. 從 `templates/chapters/*.template.tex` 複製內容
2. 貼到對應的 `chapters/*.tex`
3. 依 `specs/rules/writing-constraints.md` 完成填寫

常用對應：

- `01-introduction.template.tex` -> `chapters/01-introduction.tex`
- `02-background-preliminaries.template.tex` -> `chapters/02-background-preliminaries.tex`
- `03-proposed-algorithm.template.tex` -> `chapters/03-proposed-algorithm.tex`
- `04-illustrated-example.template.tex` -> `chapters/04-illustrated-example.tex`
- `05-experimental-results.template.tex` -> `chapters/05-experimental-results.tex`
- `06-conclusion-future-work.template.tex` -> `chapters/06-conclusion-future-work.tex`

提醒：template 是起稿骨架，不是最終內容；最終仍請回填到 `chapters/*.tex`。
